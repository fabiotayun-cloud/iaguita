CARPETA DE MIEMBROS
===================

Coloca aquí una foto de cada miembro del curso de fútbol.

El nombre del archivo (sin extensión) será el nombre del miembro.

Ejemplos:
  - Juan Perez.jpg
  - Maria Lopez.png
  - Carlos Ruiz.jpeg

Requisitos:
  - Una sola persona por foto (selfie o foto clara del rostro)
  - Formatos aceptados: jpg, jpeg, png, webp, bmp
  - El rostro debe ser claramente visible
